# Banking Management System (Core Java)

A simple console-based banking system built using Java and Object-Oriented Programming principles.

## Features:
- Create Account with Secure PIN
- Login Authentication with Attempt Limit
- Deposit and Withdraw Money
- Check Account Balance
- Transaction History using ArrayList
- Encapsulation applied for data protection

## Tech Stack:
- Java Core
- OOP Concepts (Classes, Objects, Encapsulation, Methods)
- Collections Framework (ArrayList)

## How to Run:
1. Navigate to src folder:
   cd src

2. Compile the program:
   javac com/bankapplication/*.java

3. Run the program:
   java com.bankapplication.BankApp
